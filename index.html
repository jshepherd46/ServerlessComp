About this course

Built for researchers,
by researchers

Developed at the University of Hawaiʻi Cancer Center to bridge the gap between health research and modern cloud-native infrastructure. Every exercise uses real study contexts — HIPIMR, Makawalu, HALO, PacificFed — so you're building infrastructure you'll actually use.

The paradigm throughout: sensitive data stays in a secure cloud boundary; curated containers and compute spin up beside it, do the work, and terminate. You never move PHI. You never pay for idle capacity.

Prerequisites — Comfort with command-line basics. Python experience helpful but not required. No prior cloud or Kubernetes experience assumed.

Target outcomes

  • → Spin up HIPAA-compliant GCP environments from Terraform in minutes
  • → Train and serve CNN, BERT, and LLM models on GKE & Vertex AI
  • → Deploy OMOP CDM + ATLAS on BigQuery for clinical cohort work
  • → Build federated ML across multi-site networks like PacificFed
